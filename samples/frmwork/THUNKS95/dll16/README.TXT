DLL16: Demonstrates Medium-Model DLL

DLL16 shows how to create a medium-model dynamic-link
 library (DLL) which is then thunked to from a 32-bit
 application.

DLL16 was built and tested under Microsoft(R)
Windows(TM) version 3.1.

NOTE: You should build this 16-bit dll before building
      the 32-bit application and 32-bit DLL in the directory
      up one level.
