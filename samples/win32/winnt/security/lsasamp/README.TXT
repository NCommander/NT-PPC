The APIs found in this directory are not Win32 APIs.  They are
specific APIs for Security Providers and Account Management.  
Please be aware In the future many of these APIs will be replaced
with OLE Interfaces, and will subsequently be dropped.

Documentation on ntsecapi.h can be found in LSAAPI.HLP.

Documentation on sspi.h can be found in the Service Provider Documentation,
i.e.  SPK.MVB (installed by SDK setup).
